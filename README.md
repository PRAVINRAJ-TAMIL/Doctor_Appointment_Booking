# Doctor Appointment Booking App

A Flutter-based mobile application for booking doctor appointments. This app is built using Flutter for the front-end, Firebase for backend services, GetX for state management, and features a responsive user interface for a seamless user experience.

## Features

- User registration and authentication
- Doctor profiles with details
- Booking appointments with doctors
- User and doctor dashboards

## Usage
-Register and log in as a user or a doctor.
-Browse available doctors and their profiles.
-Book appointments with your preferred doctors.
-Receive real-time notifications about your appointments.
-Access the user or doctor dashboard for managing appointments.
